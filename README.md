# Cinematico

A Rails-based movie tracking and social platform that integrates with The Movie Database (TMDb) API.

## Setup & Deployment
For end-to-end setup (local environment through production deployment), follow one of:
- [Technical Documentation](docs/TECHNICAL_DOCUMENTATION.md): comprehensive guide, architecture, and production notes
- [Contributing Guide](contributing.md): contributor-focused setup, quality gates, and Heroku deployment steps

- **[User Stories](https://github.com/tamu-edu-students/CSCE606_Fall2025_Project3_Group6/wiki/Features-and-Stories)**: Feature documentation and acceptance criteria

## Technology Stack

- Ruby on Rails 8.0.3
- PostgreSQL
- TMDb API v3
- Tailwind CSS
- Kamal (Docker deployment)
